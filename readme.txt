== Description ==

This plugins adds the attribute "loading=lazy" to all images in the posts content.
It does not add any attribute to the logo or post thumbnail images.

Logo Image by Leo from the Noun Project
Logo Image native american by sahua d from the Noun Project

== Installation ==

* Install plugin
* Activate it
